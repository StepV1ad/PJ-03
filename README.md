# PJ-03
Final project of unit 1 "Basic constructions of C++" (PJ-03)

1. Задание выполнял один. 

2. Первоначальная идея основывалась на массиве указателей, используемых для хранения чатов пользователей. Но так как данная концепция не соответствует некоторому принципу программирования, итоговая идея была взята из вебинара курса.

3. В проекте есть классы Message и User аналогичные материалу вебинара (только добавлен экземпляр id_ для работы с vector).
Используются булевы функции для проверки вводимых праметров пользователей. Раздельные методы для общего и частного чатов. Предусмотрено изменение параметров пользователя. Также добален пользователь admin со спецфункциями.
